      • THRIFT_QOP_KEY

        static final String THRIFT_QOP_KEY
        Thrift quality of protection configuration key. Valid values can be: privacy: authentication, integrity and confidentiality checking integrity: authentication and integrity checking authentication: authentication only This is used to authenticate the callers and support impersonation. The thrift server and the HBase cluster must run in secure mode.
