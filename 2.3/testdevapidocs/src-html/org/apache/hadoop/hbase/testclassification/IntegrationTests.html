021/**
022 * Tag a test as 'integration/system' test, meaning that the test class has the following
023 * characteristics:
024 * <ul>
025 *  <li> Possibly takes hours to complete</li>
026 *  <li> Can be run on a mini cluster or an actual cluster</li>
027 *  <li> Can make changes to the given cluster (starting stopping daemons, etc)</li>
028 *  <li> Should not be run in parallel of other integration tests</li>
029 * </ul>
030 *
031 * Integration / System tests should have a class name starting with "IntegrationTest", and
032 * should be annotated with @Category(IntegrationTests.class). Integration tests can be run
033 * using the IntegrationTestsDriver class or from mvn verify.
034 *
035 * @see SmallTests
036 * @see MediumTests
037 * @see LargeTests
038 */
039public interface IntegrationTests {
040}
