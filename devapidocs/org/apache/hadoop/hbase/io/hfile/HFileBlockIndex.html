org.apache.hadoop.hbase.io.hfile

Class HFileBlockIndex

  • java.lang.Object
    • org.apache.hadoop.hbase.io.hfile.HFileBlockIndex

  • @InterfaceAudience.Private
public class HFileBlockIndex
extends Object
    Provides functionality to write (HFileBlockIndex.BlockIndexWriter) and read BlockIndexReader single-level and multi-level block indexes. Examples of how to use the block index writer can be found in CompoundBloomFilterWriter and HFileWriterImpl. Examples of how to use the reader can be found in HFileReaderImpl and TestHFileBlockIndex.
