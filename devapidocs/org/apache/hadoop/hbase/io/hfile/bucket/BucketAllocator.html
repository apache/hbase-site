org.apache.hadoop.hbase.io.hfile.bucket

Class BucketAllocator

  • java.lang.Object
    • org.apache.hadoop.hbase.io.hfile.bucket.BucketAllocator

  • @InterfaceAudience.Private
public final class BucketAllocator
extends Object
    This class is used to allocate a block with specified size and free the block when evicting. It manages an array of buckets, each bucket is associated with a size and caches elements up to this size. For a completely empty bucket, this size could be re-specified dynamically. This class is not thread safe.
