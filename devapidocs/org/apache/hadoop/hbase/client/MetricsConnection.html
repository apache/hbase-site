Class MetricsConnection

  • java.lang.Object
    • org.apache.hadoop.hbase.client.MetricsConnection
  • All Implemented Interfaces:
    StatisticTrackable

    @InterfaceAudience.Private
public class MetricsConnection
extends Object
implements StatisticTrackable
    This class is for maintaining the various connection statistics and publishing them through the metrics interfaces. This class manages its own MetricRegistry and JmxReporter so as to not conflict with other uses of Yammer Metrics within the client application. Instantiating this class implicitly creates and "starts" instances of these classes; be sure to call shutdown() to terminate the thread pools they allocate.
