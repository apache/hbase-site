      • THRIFT_QOP_KEY

        static final String THRIFT_QOP_KEY
        Thrift quality of protection configuration key. Valid values can be: auth-conf: authentication, integrity and confidentiality checking auth-int: authentication and integrity checking auth: authentication only This is used to authenticate the callers and support impersonation. The thrift server and the HBase cluster must run in secure mode.
