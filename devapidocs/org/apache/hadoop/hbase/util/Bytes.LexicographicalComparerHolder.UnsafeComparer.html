        static boolean lessThanUnsignedLong(long x1,

        Returns true if x1 is less than x2, when both values are treated as unsigned long. Both values are passed as is read by Unsafe. When platform is Little Endian, have to convert to corresponding Big Endian value and then do compare. We do all writes in Big Endian format.
