Interface RSGroupInfoManager

  • All Known Implementing Classes:
    RSGroupInfoManagerImpl

    @InterfaceAudience.Private
public interface RSGroupInfoManager
    Interface used to manage RSGroupInfo storage. An implementation has the option to support offline mode. See RSGroupBasedLoadBalancer
